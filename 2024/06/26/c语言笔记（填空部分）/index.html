5、下面程序的输出结果是 。

    main( ) 
    { int m[3][3]={ {1}, {2}, {3} }；
    int n[3][3]={ 1, 2, 3 }；
        printf("%d\n", m[1][0]+n[0][0] )； 
    }

答案：3

m[0][0] = 1, m[0][1] = 0, m[0][2] = 0;
m[1][0] = 2, m[1][1] = 0, m[1][2] = 0;
m[2][0] = 3, m[2][1] = 0, m[2][2] = 0;

n[0][0] = 1, n[0][1] = 0, n[0][2] = 0;
n[1][0] = 2, n[1][1] = 0, n[1][2] = 0;
n[2][0] = 3, n[2][1] = 0, n[2][2] = 0;

所以为3
