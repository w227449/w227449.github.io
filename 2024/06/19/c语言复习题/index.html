沈阳工业大学化工过程自动化学院2024c语言复习题

1、判断字符变量c的值不是数字也不是字母，应采用下述表达式（ ）。
A) c<=48||c>=57&&c<=65||c>=90&&c<=97||c>=122
B) !(c<=48||c>=57&&c<=65||c>=90&&c<=97||c>=122)
C) c>=48&&c<=57||c>=65&&c<=90||c>=97&&c<=122
D) !(c>=48&&c<=57||c>=65&&c<=90||c>=97&&c<=122)
答案：D

2、有以下程序
#include <stdio.h>
main()
{ int y=10;
while(y–);
printf(“y=\n”, y);
}

    程序执行后的输出结果是（ ）。
    A) y=0 B) y=-1 
    C) y=1 D) while构成无限循环

答案：B

    解析：这段程序中，while(y--); 构成了一个循环。
    这个循环的条件是 y--，
    这是一个后置递减表达式，
    意味着每次循环迭代时 y 的值会先被使用，
    然后 y 的值会减少1。
    初始时，y 的值为10。
    在循环中，y 会递减直到变为0。
    由于 while 循环的条件是 y--，
    当 y 变为0时，
    循环条件不再满足，循环结束。
    但是，由于 y-- 是后置递减，
    所以即使 y 变为0，
    循环条件 y-- 会先使用 y 的当前值（0），
    然后 y 会减1，变为-1。
    循环结束后，y 的值变为-1。然后执行 printf 语句，输出 y 的值。
    所以程序执行后的输出结果是 y=-1，正确答案是 B y=-1

3、设有定义：int x＝2；，以下表达式中，值不为6的是( )。

 A) 2*x,x+=2        B) x＋＋，2*x
 C) x*＝(1＋x)      D) x*＝x＋1

答案：A

    解析：
    A) 2*x, x += 2 首先计算2*x，此时x的值为2，所以结果为4。然后执行x += 2，将x的值增加2，变为4。
    B) x++, 2*x 首先执行x++，x的原始值为2，使用这个值后x增加1变为3。然后计算2*x，此时x的值为3，所以结果为6。
    C) x *= (1 + x) 首先计算1 + x，此时x的值为2，所以结果为3。然后执行x *= 3，将x的值乘以3，变为6。
    D) x *= x + 1 首先计算x + 1，此时x的值为2，所以结果为3。然后执行x *= 3，将x的值乘以3，变为6。

4、以下定义数组的语句中错误的是( )。

    A) int num[][3]＝{ {1,2},3,4,5,6 }；
    B) int num[2][4]＝{ {1,2}，{3,4}，{5,6} }；
    C) int num[]＝{ 1,2,3,4,5,6 }；
    D) int num[][4]＝{1,2,3,4,5,6}；

答案：B

    A) int num[][3] = { {1, 2}, 3, 4, 5, 6 };
    这个语句试图定义一个二维数组，其中第二维的大小为3。数组初始化列表中包含了6个元素，可以被分为两个长度为3的子数组，这与定义的数组维度相匹配，因此这个语句是正确的。

    B) int num[2][4] = { {1, 2}, {3, 4}, {5, 6} };
    这个语句试图定义一个2行4列的二维数组。但是初始化列表中只包含了3个元素，每个元素都是一个包含两个元素的数组。这与定义的数组维度不匹配，因为2行4列的数组需要8个元素。因此，这个语句是错误的。

    C) int num[] = { 1, 2, 3, 4, 5, 6 };
    这个语句定义了一个一维数组，其中包含了6个元素。由于数组的大小没有在初始化时指定，编译器会根据初始化列表中的元素数量来确定数组的大小。这个语句是正确的。

    D) int num[][4] = {1, 2, 3, 4, 5, 6};
    这个语句定义了一个二维数组，其中第二维的大小为4。初始化列表中包含了6个元素，可以被分为两个长度为4的子数组，这与定义的数组维度相匹配，因此这个语句是正确的。

5、若有说明：int st[5]； 则对st 数组元素的正确引用是（ ）。

    A) st[5 - 1 ]    B) st[3.0] 
    C) st[2 + 3 ]    D) st( 1 )

答案：A

    解析：
    A) st[5 - 1]
    这个表达式是正确的，因为 5 - 1 的结果是4，这是数组 st 的有效索引。
    B) st[3.0]
    这个表达式是错误的，因为数组索引必须是整数值，而3.0是一个浮点数。
    C) st[2 + 3]
    这个表达式是正确的，因为 2 + 3 的结果是5，但是索引5超出了数组 st 的有效范围（0到4），所以这个引用在运行时会导致数组越界。
    D) st(1)
    这个表达式是错误的，因为数组的引用应该使用方括号 [] 而不是圆括号 ()

6、有以下程序：
#include <stdio.h>
main()
{ int a＝0，b＝0，c＝0，d＝0；
if(a＝1) b＝1；c＝2；
else d＝3；
printf(“%d，%d，%d，%d\n”，a，b，c，d)；
}
程序输出( )。

    A) 1,1,2,0   B) 0,0,0,3 
    C) 0,1,2,0   D) 编译有错

答案：D

存在一个语法错误。在 if 语句中，应该使用 == 来进行比较，而不是赋值操作符 =。正确的 if 语句应该是 if(a == 1)。
由于存在语法错误，编译器会报错，程序无法正确编译，因此不会执行，也就不会有输出。
所以，正确答案是 D 编译有错。

7、若有语句：int a=4, *p=&a；下面均代表地址的一组选项是（ ）。

    A) a,p,&*a B) &a,p,&*p 
    C) *&a,&a,*p D) *&p,*p,&a

答案：B

    解析：
    a 是变量的值，不是地址。
    p 是指针变量，它存储的是变量 a 的地址。
    &*a 是先取 a 的地址，然后解引用该地址，得到 a 的值，不是地址。
    *&a 是先取 a 的地址，然后解引用该地址，得到 a 的值，不是地址。
    &a 是变量 a 的地址。
    *p 是解引用指针 p，得到 a 的值，不是地址
    *&p 是先取指针 p 的地址，然后解引用这个地址，这将得到 p 指针本身的值，即变量 a 的地址，这是一个地址。
    *p 是解引用指针 p，得到 a 的值，不是地址。
    &a 是变量 a 的地址。

8、设变量a是int型，f是float型，i是double型，则表达式10+’a’+i*f值的数据类型为（ ）。

    A) char   B) int 
    C) float  D) double

答案：D
解析：
char 和 short int 类型的值在参与运算前会被提升到 int 类型。
如果运算中存在 float 类型的值，int 类型的值会被提升到 float 类型。
如果运算中存在 double 类型的值，其他数值类型的值都会被提升到 double 类型。
如果有更复杂的类型混合，比如 long int、long float 等，会根据它们在类型提升层级中的位置来确定最终的运算结果类型。
9、下面程序的运行结果是（ ）。
a=1； b=2 ； c=2；
while (a<b<c) { t=a ； a=b ； b=t； c– ；}
printf (“%d , %d, %d”, a, b, c)；

    A) 2, 1, 0      B) 2, 1, 1 
    C) 1, 2, 0      D) 1, 2, 1

答案：C

    解析：一开始满足循环条件，因此进入循环，a，b调换值，执行循环过程中c值不变，此时c值为2，因此再次进入循环，循环进行中c为1，不符合再次进入循环条件，跳出循环，循环结束后c为0，因此a，b值再次调换，即为原值。

10、若x,i,j和k都是int型变量，则执行下面表达式后x的值为（ ）。
x=(i=4,j=16,k=32)
A) 4 B) 16
C) 32 D) 52
答案：C

    解析：根据逗号运算符的规则，整个表达式 (i=4,j=16,k=32) 的结果将是最后一个赋值表达式 k=32 的结果，即 k 的值。因此，x 将被赋值为 32。

11、sizeof(float)是（ ）。

    A) 一个双精度型表达式 B) 一个整型表达式
    C) 一种函数调用 D) 一个不合法的表达式

答案：B

以下选项中正确的输入语句是（ ）。

13、设有定义：long x=-123456L； 则以下能够正确输出变量x值的语句是（ ）。
