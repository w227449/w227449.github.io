课时1——c语言基础知识

1.基本常识

一个c语言程序有且只有一个main函数，是程序运行的起点。每次c程序写完后都是先编译，后链接，在运行

2.标识符的概念

a.标识由由字母，数字，下划线组成
b.第一个必须为字母/下划线
c.标识符分为关键字，预定义标识符，用户标识符等
d.关键字：不可以做为用户标识符

 If可以做标识符，I大写了，非关键字

e.预定义标识符：可做用户标识符

 背诵define （定义）
 scanf（输入）
 printf（输出）
 include（引用的头文件）

f.用户标识符：用户自己命名的
关键字列表：

一、数据类型关键字

 char        声明字符变量
 double      声明双精度变量
 float       声明浮点型变量
 int         声整型变量
 short       声明短整型变量

二、控制语句关键字

（1）循环语句关键字（5个）

 for      遍历循环
 do       其后紧跟循环体

  else    条件语句否定分支

 switch  用于多条件判断语句

 default 开关语句的其他分支

（4）开关语句关键字（1个）

 return  函数返回语句

三、储存类型关键字

 auto    声明自动变量
 extern  声明变量是1在其他文件的定义1
 register 声明寄存器变量
 static   声明静态变量

四、其他关键字

const    声明只读变量
sizeof   计算数据类型长度（字节数）
typedef  给数据类型取别名
volatile 所修饰的对象不能被编译器转化

3.进制转化

（1）c语言中只有八，十，十六，没有二进制
（2）八进制以0开头，没有8，遇到8进1
（3）小数点的合法写法：
EP：1.0写成1. / 0.1写成。1
（4）实行数据合法写法：

EP： 2.3333e-1合法数据为2.333x110-1

 # 考试口诀：e前e后必有数，e后为整数

（5）整形4字节，字符型1字节，双精度型8字节
float 4字节
double 8字节
char 字符型
字符串最少2字节

4.算术表达式与字符表达式

（1）算是表达式 +，-，*，/，%
a. “/”两边为整形结果为整形
b. “/”有一边为小数， 结果为小数
c. “%”符号请异地内阁要注意是余数，%两边要求为整数
（2）赋值表达式
表达式数值是右边的数值，a=b=5；该表达式为5
常量不可以赋值
（3）符合赋值表达式
算术运算符与“=”结合在一起，形成复合赋值运算符
（4）自加表达式
a=5 ++a(为6) a++(为5)

# 考试口诀：++在前先加后用，++在后先用后加

（5）逗号表达式
优先级最低
Ep：z=（2，3，4） z=4
Z=2，3，4 z=2

【3】强制类型转换
一定是（int）a，不是int（a）类型上一定有括号

【4】三种取整丢小数的情况
int a =1.6

1/2，3/2

5.字符

（1）字符数据的合法形式：

a. ’1’是字符占一个字节，“1”是字符串占两个字节（含有一个结束的符号）。
b. ’0’的ASCII数值标识为48，’a’的ASCII数值标识为97，’A’的ASCII数值标识65

一般考试表示单个字符错误的形式’65’ “1”
字符是可以进行算术运算的，记住’0’-0=48
大写字母和小写字母转化的方法 ’A’+32=’a’ ，相互之间一般相差32.
’A’单引号只占一个字节
“A”双引号占两个字节，其中有一个结束符号

（2）转义字符
a.转义字符分为一般转义字符，八进制转义字符，十六进制转义字符。

【1】一般转义字符：背诵

   \t    \n    \'   \"   \\;

【2】八进制转义字符

 '\141'是合法的，前导的0是不能写的

【3】十六进制转义字符

 '\x6d'才是合法的，前导的0不能写，并且x是小写

常用转义字符的及应用

字符形式 含义 ASCII码
\n 换行 10
\t 水平制表（下一个Tab） 9
\b 退格 8
\r 回车 13
\f 换页 12
// 反斜杠\ 92
' 单引号（撇） 39
" 双引号 34
\ddd 1到3位八进制代表的字符
\xhh 1到2位3十六进制代表的字符

(3)字符型和整型是近亲：两个都具有很大的相似之处
char = 65 ；
printf（“%c”,a）,得到输出结果：A
printf（“%d”,a）,得到输出结果：65

？转义字符最常用的功能：控制输出的格式
示例

    #include<stdio.h>
    void main()
    {
    char c1 ='A',c2='N';
    printf("a\tbc\tABC\n);
    printf("xyz\t%c %c",c1c2);
    }
输出结果如下
      a      bc    ABC
      xyz    AN

课时二 输出输入函数

1.数据输出printf()

(1)使用printf和scanf时，要在最前面加上#include<stdio.h>
(2)printf可以只有一个参数，也可以有两个参数(选择题)
(3)printf(“第一部分”，第二部分)；把第二部分的变量，表达式，常量以第一部分的形式表达出来！
(4)printf(“a=%d,b=%d”，12，34) 重点！
一定要记住是将12和34以第一部分的形式展现在终端上
printf(“a=%d,\n b=%d”，12，34)
因为加入了\n所以2输出结果是
a=12，
b=34
(5)int x=017;一定要弄清楚！

      printf("%d,x");八进制转换为十进制输出为15
      printf("%o,x");输出为八进制：17   八进制
      printf("%#o,x")；输出为八进制：017（带八进制前导）
      printf("%x,x");输出为十六进制：f   十六进制
      printf("%#x,x")；输出为十六进制：017（带十六进制前导）

常用printf转换描述

格式说明 表示内容 格式说明 表示内容
%d 以整数形式输出 %c 以字符形式输出
%ld 长整型long int %s 字符串
%f 浮点型float %o 以八进制输出
%lf double %#c 带前导的八进制
%% 输出一个百分号 %x 以十六进制输出
%5d %#x 带前导的十六进制
转换描述 对应参数类型 实现的转换
:-: :-: :-:
%md int 输出十进制整数，m为输出最小宽度（如数据位的小数小于m，则左端补空，若数据位数大于m，则按实际长度输出）
%m.nf float,double 按带小数形式输出实数，m为输出最小宽度，n为小数点后位数。（如数据的位数小于m，则左端补空，若数据位数大于m，按实际长度输出。小数位数被截断时按四舍五入输出）

2.数据输出scanf

(1)scanf(“a=%d,b=%d”,&a,&b) 

 考试的超级重点
 一定要记住是以第一部分的格式在终端输入数据。
 在屏幕上显示的输入为 a =12；，b=34才可以把12和34正确的给a和b。有一点不同也不行

(2)scanf(“%d,%d,x,y”);

这种写法是绝对错误，scanf的第二部分一定要是地址！

(3)特比注意指针在scanf的考察
EP： 

int x=2；int *p；
p&x
scanf("%d",x);  错误
scanf("%d",p);  正确
scanf("%d",&p); 错误
scanf("%d,*p);  错误
加&取地址，加*取数值

(4)指定输入的长度(考试重点)

 终端输入：1234567
 scanf("%2d%4d%d",&x,$y,&z),x为12，y为4567，z为7
 终端输入：1 2345667 由于1和2有空格，所以只有一位给x
 scanf("%2d%4d%d",&x,$y,&z);x为1，y为2456，z为667

(5)考试的超级重点：输入时候字符和整数的区别

 scanf("%d",&x);这时候输入1，特别注意表示的是整数1
 scanf("%c",&x);这时候输入1，特别注意表示的是字符“1”ASCII为整数49

补充说明

(1)scanf函数格式的考察

 注意该函数的第二部分是&a，这样的地址不是a
 scanf("%d%d*d%d",&x,$y,&z)跳过输入的第三个数据

(2)putchar,getchar函数的考察 

 char a = getchar()是没有参数的，从键盘得到你输入的一个字符变量给a
 putchar('y')把字符输出到屏幕上面

(3)如何实现两个变量x和y中数值的互换（背诵）

 不可把x=y，y=x；要用中间变量 t=x；x=y；y=t；
