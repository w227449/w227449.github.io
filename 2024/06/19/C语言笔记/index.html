一.表达式

1.关系表达式

a.关系表达式的数值只能为1(表示为真)，或0(表示为假)

  如9>8这个关系表达式为真的，所以9>8这个表达式的数值为1.
  如7<6这个关系表达式为假的，所以7<6这个表达式的数值为0.

b.考试最容易错的：就是int x=1，y=0，z=2；

  x<y<z是真的还是假的？
  带入为1<0<2,
  从数学的角度出发肯定是假的，
  但是如果是c语言就是正确的！
  1<0是假的得到0，
  表达式就变成了0<2那么运算结果就是1

c.等号和赋值的区别！

  一定要记住“=”就是赋值，“==”才是等号

2.逻辑表达式:

 核心：表达式的数值只能为1(表达为真)或0(表示为假)。

 (a)共有&&  || ！三种逻辑运算符号
 (b)！> && > ||   优先级的区别！
 (c)表示x大于0还是小于10的方法

     0<x<10是不行的(一定记住)。是先计算0<x得到结果是1或0；再用0或1与10比较得到总是真（为1）。所以一定要用(0<x)&&(x<10)表示x在0到10之间

课时五 指针

  指针变量唔知是用来放地址的，而一般的变量是放数值的。

1.指针的定义 两种形式

      (1)int x=3;int*p=&x
      (2)int x; int*p
      x=3;p=&x
      int *p 中 *p和p的差别：简单来说*p是数值，p是地址！
      *p可以当作变量来使用；*的作用是取后面地址p里面的数值
      p是当地址来使用。可以用在scanf函数中：scanf("%d",p)

2.*p++和(*p)++之间的差别：该做题目中重要！考试的超级重点

      *p++是地址会变化。 
      口诀：取当前值，然后在移动地址！
      (*p)++是数值会要发生变化。 
      口诀：取当前值，然后在使数值加1

3.二级指针

      *p：一级指针：存放变量的地址
      **q：二级指针：存放一级指针的地址

                常考题目：
                  int x =7；
                  int*p=&x，**q=p；
                  问：*p为7，*q为p，**q为7；

4.三名注意（考试的重点）

      数组名：表示第一个元素的地址。数组名不可以自加，他是地址常量名字。
      函数名：表示该函数的入口地址
      字符串常量名：表示第一个字符的地址

5.移动指针(经常加入到考试中的其他题目综合考试)

      char *s =“meikanshu”
      while (*s){printf("%c",*s);s++;}
      这个s首先会指向第一个字母m然后通过循环会第一次打印出一个字符，s++是地址移动，打印了一个字母后，就会移动到下一个字母！

      方法二：int a =2，*p；(定义之后的初始化)

课时6 数组

      数组：存放的类型是一致的。多个数组元素的地址是连续的。
      最大的特点：一次存入，多次使用。

1.一堆数组的初始化

      int a[5]={1,2,3,4,5};         合法
      int a[5]={1,2,3,   };         合法
      int a[ ]={1,2,3,4,5};         合法
        #int a[5]={1,2,3,4,5,6};    #不合法

2.一维数组定义：

      int a[5];注意这个地方有一个重要的考点，定义时数组的个数不是变量一定是常量。
      
      int a[5]                     合法
      int a[1+1]                   合法
      int a[1/2+4]                 合法
      int x =[5],int a[x];         不合法
      define p 5 int a[p]          合法，define后的p是符号常量。

二位数组的初始化

      int a[2][3]={1,2,3,4,5,6}；     合法，很标准的形式
      int a[2][3]={1，2，3，4，5 }；   合法，后面默认为0
      int a[2][3]={{1，2，3}{4，5，6 }}；合法，，每行3个
      int a[2][3]={{1，2， }{4，5，6 }}；合法，第一行最后一个默认为0；
        int a[2][3]={1,2,3,4,5,6,7}；     不合法
      int a[ ][3]={1,2,3,4,5,6}；        合法，可以缺少行不可以缺少列
      int a[2][ ]={1,2,3,4,5,6}；        不合法

补充

一堆数组的重要概念：

对a[10]这个数组的讨论。

      1.a表示数组名，是第一个元素的地址，也就是元素a[0]的地址。等价于&a
      2.a是地址常量，所以只要出现a++，或者是a=a+2赋值都是错误的
      3.a是一维数组名，所以他是列指针，也就是说a+1跳一列

对a[3][3]的讨论

      1.a表示数组名，是第一个元素的地址，也就是元素a[0][0]的地址。
      2.a是地址常量，所以只要出现a++，或者是a=a+2赋值都是错误的
      3.a是二维数组名，所以他是行指针，也就是说a+1跳一行
      4.a[0],a[1],a[2]也都是地址常量，不可以对他进行赋值操作，同时他们都是列指针，a[0]+1,a[1]+1,a[2]+1,都是跳一列。
      5.注意a和a[0],a[1],a[2]是不同的，他们的基本类型不同。前者是一行元素，后三这是一列元素。

二维数组做题目的技巧

      如果有a[3][3]={1，2，3，4，5，6，7，8，9}这样的题目
      *步骤一：把他们写成：
                 第一列        第二列         第三列 
           a[0]    1             2             3
           a[1]    4             5             6
           a[2]    7             8             9
       步骤二这样的题目操作很简单
         *(a[0]+1)我们就知道是第一行的第一个元素往后面跳一列，那么这里就是a[0][1]元素，所以是2
         *(a[1]+2)我们就知道是第二行的第一个元素往后面跳两列，那么这里就是a[1][2]元素，所以是6.
         一定记住：只要是二维数组题目，一定写成如上格式，再去做题目，这样比较简单。

记住以下表达形式

      a[2]变成*(a+2)
      a[2][3]变成*(a+2)[3]，然后可以变成*(*(a+2)+3)
      这个思想很重要！
